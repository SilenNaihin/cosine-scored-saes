# Experiment 37: Goldilocks/regularized training variants

Systematic comparison of 20 regularization approaches for per-feature scale_a on Gemma-2-2B Layer 13. Three strategies tested: L2 penalty on scale_a (lambda in {0, 0.01, 0.1, 1.0, 10.0}), group-wise sharing (G in {1, 4, 16, 64, 256, 9216}), and scale_a dropout (p in {0, 0.1, 0.3, 0.5, 0.7, 0.9}). All runs use sqrt(d) initialization, 50M tokens, d_sae=9216, k=80.
